simple cli tool to mass convert all aseprite project slices out as standalone images. Your aseprite(.exe) is expected to be in your system path. (meaning you can start aseprite by opening a CLI and typing `aseprite` or `aseprite.exe`)

Normal people would make a bat file, WSL a shell script, powershell or just use python. Im just trying to get shit done with what i like working it.

publishing this as a single file and also adding it to your path, then making a bat file containing only `SliceAseprite.exe` (the name of this programs exe) will give you an easy to click way of starting this, or just type it in the cli
